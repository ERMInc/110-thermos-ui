THERMOS Help

Keyboard shortcuts

  • c - rotate the Constraint status of the selection
  • z - zoom to fit the selection or document
  • a - select everything in the document
  • t - change the Type of a building

Concepts

THERMOS is a program to help you think about heat networks. It is based around two main parts:

  1. A spatial database containing buildings and roads. THERMOS calls these candidates. They are the ingredients it can consider when designing a heat network.
  2. An optimisation model, which can determine the quality of a network made up of some candidates, and can construct a network of good quality by choosing some candidates.
